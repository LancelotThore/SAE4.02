Eco Quest

Concept Lore Commandes

Le concept

EcoQuest est un jeu autour de l’écologie dans lequel le joueur doit trouver des déchets, caché sur la map, qui est une île déserte. Le but est donc de trouver tous les déchets afin de nettoyer l’île.

Notre but est de faire comprendre aux gens à quel point c’est important de prendre soin de notre planète. Chaque déchet ramassé dans le jeu représente un petit pas vers un avenir plus durable. Le jeu invite les joueurs à réfléchir à leur propre impact sur l'environnement et à prendre des mesures concrètes pour réduire leur empreinte écologique.

Le Lore

Vous démarrez votre partie sur un bord de l’île, et vous découvrez un perroquet qui vous explique le problème : il y a peu, un bateau s’est échoué prêt de l’île et ses passagers, en attendant les secours, on laissé pleins de déchets !

Vous devez tous les ramasser pour que votre ami le perroquet et les autres animaux retrouvent leur terrain de jeu.

Les Commandes

Ce jeu peut se jouer en VR ou sur votre navigateur ! Il est très simple d’utilisation.

En VR : vous vous déplacez avec le joystick gauche et vous bougez la caméra avec le joystick droit ou tout simplement avec votre tête. Pour interagir avec les objets ou les animaux, vous devez mettre le curseur sur l’objet et appuyer sur votre gâchette.

Sur un navigateur : vous vous déplacez avec les touches ZQSD et vous bougez votre caméra avec votre souris, en vue à la première personne. Pour interagir, vous avez juste à viser et appuyer sur le clic gauche de votre souris.

Le Code

Les déchets

a-entity id="dechets" : c'est un conteneur pour les objets liés aux déchets. Il est utilisé pour regrouper des éléments (ici les déchets).

a-entity : génération d’un déchet.

- id="trash object" : l’id object est utilisé pour le score.

- position : définit la position de l'entité dans l'espace 3D.

- obj-model : charge un modèle 3D à partir de fichiers .obj et .mtl.

- scale : spécifie l'échelle de l'entité dans les trois dimensions.

- event-set__down : définit une action à effectuer lorsqu'un événement "mousedown" se produit.

- event-set__enter : définit une action à effectuer lorsque le curseur entre dans la zone de l'entité.

- event-set__leave : définit une action à effectuer lorsque le curseur quitte la zone de l'entité.
